三层工程体系

DESIGN PHILOSOPHY

为什么需要这套体系？安全团队面对的不是单一漏洞，而是持续涌入的告警噪声、跨系统的漏洞扩散、难以闭环的修复验证。传统方案只能点状响应，无法从全局视角驱动治理。 我的思路是：先打通底层框架的流式与并行能力，让 Agent 具备生产级可靠性；再构建可观测的平台层，让复杂的多 Agent 协作对工程师透明可调；最终在应用层实现漏洞全生命周期的智能闭环——从代码提交到云上运行时，每个环节都有 Agent 在守护。

02

核心项目详情

框架层 · 基础设施

Agno 框架改造

在多 Agent 框架尚不成熟、专用 Agent 模型尚未出现的时代，对 Agno 框架进行深度改造，成为整个体系的技术地基。

  • reasoning_content 流式输出——将推理过程实时暴露给前端
  • 并行工具与 Agent 调用——大幅降低多 Agent 协作延迟
  • 子 Agent 流式输出——支持 toolcall/reasoning 事件透传
  • 全异步改造——提升并发吞吐量
  • spawn_agent——运行时动态创建子 Agent，支持弹性任务分解
平台层 · 告警分析

告警分析多 Agent 系统

分层编排的安全告警智能研判系统，针对海量重复告警设计了快速/深度双模式分析策略。

  • 分层架构：主 Agent + 日志分析 Agent（DNS/HIDS/EDR 等子 Agent）+ 知识检索 Agent
  • 快速研判：基于历史报告快速判断，跳过深度调查，1-2 分钟出结果
  • 上下文注入：注入近期告警列表，自动识别重复告警避免重复分析
  • 深度分析：复杂告警全链路分析，6 分钟内完成
1-2min快速研判
6min深度分析
平台层 · 可观测性

透明化 Agent 前端

不只是交互界面——核心定位是 Agent 行为可观测平台，将 Agent 从黑盒变为可调试、可信任的透明系统，加速迭代优化。

  • 嵌套 Agent 全链路追踪：可查看会话中所有 Agent（含嵌套子 Agent）的全过程，包括 reasoning 和 toolcall
  • IDE 面板：Agent 修复代码后，支持查看代码变更并一键 apply GitLab 变更
  • 流式实时渲染：工具调用流式支持，前端可实时流式渲染 UI，体验无卡顿
  • 系统定位创新：将 Agent 前端重新定义为「观测与调试」工具，而非单纯「交互」工具
产品层 · 本地工具

本地代码安全审查 Skill

从 Skill 演进为 Plugin，用户本地安装即可使用的完整代码安全审查工具，具备完整数据闭环。

  • 多维审查：代码漏洞 + SCA 依赖分析 + API 接口 + 污染链漏洞
  • 本地数据闭环：所有扫描记录结构化存储本地，含 meta 和漏洞信息，支持历史查询与漏洞去重
  • 修复决策 hook：PostToolCall hook 触发修复，遥测数据上报云端接力
  • 灵活扫描范围：full 全量 / diff 最新 commit / file 单文件
  • 完整闭环：漏洞审查 → 修复 → 测试用例生成
产品层 · Codian

Codian 安全代码平台

面向全公司研发人员的 AI 安全代码审查平台，覆盖从 MR 自动审查到对话式漏洞分析的完整工作流。

  • GitLab OAuth 认证：企业级权限管理
  • Webhook 一键订阅：MR 事件自动触发 Agent 代码审查
  • 对话式漏洞分析：选择仓库和分支，流式返回分析结果
  • 代码差异 Apply：一键应用 AI 生成代码变更并自动创建 MR
  • 本地遥测数据展示：集成 Skill 上报的 repo commit 扫描漏洞详情

03

构建历程

Phase 01 · 框架先行
Agno 框架深度改造
在没有成熟 Agent 框架、没有专用 Agent 模型的时代率先入场。改造 Agno 以支持流式推理、并行调用、动态子 Agent，为后续所有项目奠定技术底座。
Phase 02 · 平台构建
多 Agent 后端 + 透明化前端
基于改造后的框架，构建告警分析多 Agent 系统（分层编排、双模式研判），同步开发透明化 Agent 前端，将 Agent 运行行为完全可观测，实现快速调试迭代。
Phase 03 · 产品化
Codian + 本地审查 Skill
将平台能力产品化：Codian 面向全公司研发落地 MR 审查和对话式分析；本地 Skill 将能力延伸至开发者本地环境，构建遥测→云端接力的完整数据闭环。
Phase 04 · 规模化
5+ 安全 Agent 批量上线
平台能力成熟后，云上漏洞分析、Skill 安全审查、API 安全审查等多个场景 Agent 快速接入上线。新 Agent 的落地成本从「从零搭建」降低为「设计 Prompt 和工具」。
